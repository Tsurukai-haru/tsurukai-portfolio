# tsurukai-portfolio

個人ポートフォリオサイト。日本語・英語の切り替えに対応しています。

**公開URL:** https://tsurukai-haru.github.io/tsurukai-portfolio/

## 技術スタック

- [Next.js](https://nextjs.org/) 16 (Static Export)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) v4

## ローカルで動かす

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## デプロイ

`master` ブランチに push すると GitHub Actions が自動でビルド・デプロイします。
